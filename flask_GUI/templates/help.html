Help

Configuration Name: A configurations file with this name will be saved at "...\statistics_tool\configs"

File Reading Function: A function that responsible for a few aspects of reading the input files, in order to implement a new one follow the instructions in "...\statistics_tool\user_defined_functions\ReadingFunctions.py"

Overlap Function: An overlap function (e.g IOU, DICE), in order to implement a new one follow the instructions in "...\statistics_tool\user_defined_functions\OverlapFunctions.py"

Threshold: A number between 0-1 that indicates the minimum overlap between a prediction and a label to be considered a TP

Evaluation Function: A function that match predictions and labels, in order to implement a new one follow the instructions in "...\statistics_tool\user_defined_functions\EvaluationFunctions.py"

Statistics Functions: A statistical function (e.g Precision, Recall), in order to implement a new one follow the instructions in "...\statistics_tool\user_defined_functions\StatisticsFunctions.py"

Partitioning Functions: A function that partition the data into groups (e.g Day/Night, Car/Bus/Motorcycle), in order to implement a new one follow the instructions in "...\statistics_tool\user_defined_functions\PartitioningFunctions.py"
